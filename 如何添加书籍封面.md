# 📚 如何为每本书添加不同的封面

## 方法一：使用 GitHub 存储图片（推荐）

### 步骤1: 准备图片
将书籍封面图片准备好，建议：
- 格式：JPG 或 PNG
- 尺寸：建议高度 400-600px
- 文件名：使用英文，如 `book1.jpg`, `book2.jpg`

### 步骤2: 上传到 GitHub
1. 在你的 GitHub 仓库创建 `images` 文件夹
2. 将封面图片上传到这个文件夹
3. 点击图片，复制浏览器地址栏的 URL
4. **重要**：在 URL 末尾加上 `?raw=true`

**示例：**
```
原始链接：
https://github.com/用户名/仓库名/blob/main/images/book1.jpg

加上 ?raw=true：
https://github.com/用户名/仓库名/blob/main/images/book1.jpg?raw=true
```

### 步骤3: 修改 books.ts
打开 `src/config/books.ts`，找到对应的书籍，填入图片链接：

```typescript
{
  id: 1,
  title: '书籍标题 1',
  author: '作者名称',
  cover: 'https://github.com/用户名/仓库名/blob/main/images/book1.jpg?raw=true', // ⬅️ 填入图片链接
  rating: 5,
  readDate: '2024-01',
  review: '这是一本很棒的书...',
  tags: ['小说', '科幻']
}
```

## 方法二：使用本地图片（最简单）

### 步骤1: 创建文件夹
在项目根目录创建 `public/images/books/` 文件夹

### 步骤2: 放入图片
将书籍封面图片复制到这个文件夹，例如：
```
public/
  images/
    books/
      book1.jpg
      book2.jpg
      book3.jpg
      ...
```

### 步骤3: 修改 books.ts
使用相对路径引用图片（注意开头的 `/`）：

```typescript
{
  id: 1,
  title: '书籍标题 1',
  author: '作者名称',
  cover: '/images/books/book1.jpg', // ⬅️ 使用相对路径
  rating: 5,
  readDate: '2024-01',
  review: '这是一本很棒的书...',
  tags: ['小说', '科幻']
},
{
  id: 2,
  title: '书籍标题 2',
  author: '作者名称',
  cover: '/images/books/book2.jpg', // ⬅️ 不同的图片
  rating: 4,
  readDate: '2024-02',
  review: '值得一读的好书...',
  tags: ['技术', '编程']
}
```

## 方法三：使用在线图床

### 推荐的免费图床：
- **Imgur**: https://imgur.com/
- **SM.MS**: https://sm.ms/
- **ImgBB**: https://imgbb.com/

### 使用步骤：
1. 访问图床网站
2. 上传书籍封面
3. 复制图片直链
4. 在 `books.ts` 中填入链接

## ⚠️ 注意事项

### 1. 如果使用新的图片域名
需要在 `next.config.mjs` 中添加域名配置：

```javascript
images: {
  unoptimized: true,
  remotePatterns: [
    // ... 现有配置
    {
      protocol: 'https',
      hostname: '新图片网站域名.com'  // ⬅️ 添加新域名
    }
  ],
}
```

修改后需要**重启开发服务器**。

### 2. 图片建议
- 推荐使用 3:4 的纵向图片（如 300x400、600x800）
- 文件大小控制在 500KB 以内
- 使用 JPG 格式可以减小文件大小

### 3. 没有封面的处理
如果某本书暂时没有封面，可以：
- 留空：`cover: ''`
- 系统会显示一个书本图标占位

## 🎯 快速测试

修改后保存文件，访问：
```
http://localhost:3000/reading
```

查看效果！如果图片没显示，检查：
1. 图片链接是否正确
2. GitHub 链接是否加了 `?raw=true`
3. 是否需要重启开发服务器

## 💡 小技巧

### 批量重命名图片
如果有很多图片需要重命名，可以使用：
- Windows: 选中文件 → F2 → 输入 `book` → Enter（会自动编号）
- Mac: 选中文件 → 右键 → 重新命名

### 豆瓣获取封面
1. 在豆瓣读书搜索书名
2. 右键点击封面 → "在新标签页中打开图片"
3. 复制图片链接使用
